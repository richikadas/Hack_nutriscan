## 🧠 About the Project

**Problem Statement:**
There is a need for a simple and accessible solution that allows users to quickly identify food products and understand their ingredients, calories, and nutritional values, enabling them to make informed and healthier dietary choices.

**NutriScan** is a mobile application that allows users to scan food products using images, QR codes, or barcodes to instantly access detailed nutritional information. Built in just 24 hours during Hack Horizon 2.0, the app simplifies nutrition awareness and empowers users to make smarter food decisions.

**Goal:**
To provide users with real-time insights into food products, including ingredient analysis, calorie count, and health recommendations such as whether a product is safe, should be consumed in moderation, or avoided based on dietary needs and health conditions.

**Who is it for?**
NutriScan is designed for everyone — from health-conscious individuals and fitness enthusiasts to people with dietary restrictions, allergies, or medical conditions.

## ✨ Features

* 🔍 Smart Food Scanning — Scan food items via image, barcode, or QR code
* 📊 Nutritional Insights — Get detailed information on calories, ingredients, and nutrients
* ⚠️ Health Recommendations — Know if a product is safe, moderate, or risky to consume
* 🧠 Dietary Awareness — Helps users consider allergies, conditions, and diet plans
* 📱 User-Friendly Interface — Simple and intuitive mobile app experience
* 🔔 Real-Time Results — Instant analysis using integrated databases

## ⚙️ Getting Started

### Prerequisites

* Node.js / Python installed
* Git installed
* Mobile emulator or physical device

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/your-username/nutriscan.git

# 2. Navigate into the project folder
cd nutriscan

# 3. Install dependencies
npm install
# OR
pip install -r requirements.txt

# 4. Run the project
npm start
# OR
python app.py
```
